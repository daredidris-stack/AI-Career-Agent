# AI Career Agent

An AI-powered career assistant built with Python and Ollama.

## Features

- AI Career Advice
- Resume Analysis
- PDF Resume Reader
- DOCX Resume Reader
- ATS Resume Scoring
- Job Matching
- Career Ranking
- Skill Gap Analysis
- Personalized Learning Plan
- PDF Report Export
- Word Report Export

## Technologies

- Python
- Ollama
- Qwen 3 8B
- ReportLab
- python-docx
- pdfplumber

- Live Job Search (The Muse)
